Sistema de controle de materiais

A Empresa XPTO SA necessita de um software para realizar controle de entradas/saídas e localização de materiais imobilizados e/ou de consumo direto.

Será desenvolvimento um sistema WEB utilizando tecnologia Microsoft .Net com banco de dados SQL Server.

O Sistema deverá ter uma base de cadastro de funcionários, que cria um usuário com uma senha padrão, quando um novo funcionário é contratado.

O sistema deverá ter uma funcionalidade para que os funcionários que esqueceram da senha, consiga reseta-las mediante respostas de algumas perguntas.

O Sistema deverá ter a possibilidade de cadastro manual de material e de cadastro em massa (via carga de arquivo), pois após a implantação em produção, iremos carregar toda a base de dados atual no sistema.

O sistema deverá ter integração com o sistema de fiscal, para que no momento da entrada da NF de novos materiais, o estoque atual do material seja atualizado.

O sistema deverá ter integração com o sistema de compras, para que no momento do planejamento de novas compras, já seja considerado se existe alguma compra em andamento.

O sistema deverá ter integração com o cadastro de funcionários, pois equipamentos consideramos imobilizados, deverão ser vinculados a matricula do funcionário que está utilizando.

O sistema deverá permitir que os funcionários façam solicitação de materiais, mediante disponibilidade em estoque.

O sistema deverá permitir que o almoxarife realize a liberação dos materiais solicitados, e caso seja imobilizado, realize o vinculo do equipamento liberado na matricula do funcionário que solicitou.

O sistema deverá permitir ao Gestor que realize alteração das quantidades em estoque, para garantir a acuracidade nos inventários realizados. Todas as alterações deverão ser guardadas em log para futuras auditorias,

O sistema deverá ter integração com a Sefaz para emissão de nota de simples remessa para realizar transferência de materiais entre filiais interestaduais.

O sistema deverá gerar relatórios para o Gestor acompanhar a evolução de saída/entrada dos equipamentos.

O sistema deverá ter um cadastro de planejamento de materias, para garantir que não haja falta em estoque.

O sistema deverá emitir um alerta quando um determinado material atingir um determinado % do estoque.
