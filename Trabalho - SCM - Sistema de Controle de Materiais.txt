Sistema de controle de materiais

A Empresa XPTO SA necessita de um software para realizar controle de entradas/saídas e localização de materiais imobilizados e/ou de consumo direto.

Será desenvolvimento um sistema WEB utilizando tecnologia Microsoft .Net com banco de dados SQL Server.

O Sistema deverá ter uma base de cadastro de funcionários, que cria um usuário com uma senha padrão, quando um novo funcionário é contratado.

O sistema deverá ter uma funcionalidade para que os funcionários que esqueceram da senha, consiga reseta-las mediante respostas de algumas perguntas.

--> O sistema deverá conter um serviço de recuperação de senha, mediante a confirmação de alguns dados cadastrais.

O Sistema deverá ter um cadastro de material e uma possibilidade de integração com outros sistemas (Compras, Fiscal via carga de arquivo), pois após a implantação em produção, iremos carregar toda a base de dados atual no sistema.

O sistema terá no ato da entrada da NF de materiais a atualização da guantia dos estoques.

O sistema terá no ato do planejamento de novos pedidos de compras, a verificação se existe alguma pedido em andamento.

O sistema deverá permitir que os funcionários façam solicitação de materiais, mediante disponibilidade em estoque e vinculará o cadastro do funcionario ao registro do equipamento solicitado.

O sistema deverá permitir que o almoxarife realize a liberação dos materiais solicitados.

O sistema deverá permitir ao Gestor realizá alteração das quantidades em estoque, para garantir a acuracidade nos inventários realizados. Todas as alterações deverão ser guardadas em log para futuras auditoria.

O sistema deverá ter integração com a Sefaz para emissão de nota de simples remessa para realizar transferência de materiais entre filiais interestaduais.

O sistema deverá gerar relatórios para o Gestor realizar o acompanhamento e a evolução de saída/entrada dos equipamentos.

O sistema deverá ter um cadastro de planejamento de materias, para garantir que não haja falta em estoque.

O sistema deverá emitir um alerta quando um determinado material atingir um determinado % do estoque.
